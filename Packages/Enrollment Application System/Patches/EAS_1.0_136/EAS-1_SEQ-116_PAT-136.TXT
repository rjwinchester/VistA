Subject: ENROLLMENT SYSTEMS (ES) 4.5 RELEASE 

Category: 
  - Informational

Description:
============

 The purpose of this Informational patch is to announce the release of the
 Enrollment System (ES) 4.5.
  
 The Enrollment Health Benefits Determination Program (EHBD) provides
 enhancements to the Enrollment System.
  
 This ES 4.5 release, developed in Java technology, will have Enrollment
 System Modernization (ESM) development efforts, including enhancements to
 the ES application and Web Service changes to begin utilizing the Enterprise
 Military Information Services (eMIS):
  
 Military Service Data Sharing (MSDS) to Enterprise Military Information
 Services (eMIS) Conversion
 -Migration from MSDS broker to eMIS as the authoritative source for military
  service data to ES
  
 Manage Reason Eligibility Status is Pending Verification
 - Add "Reason Eligibility Status is Pending Verification" field to the
   Eligibility Tab
 - Add "Reason Eligibility Status is Pending Verification" required field
   check to the Complete the Registration process (Add a Person)
  
 Add or Edit Temporary Mailing Addresses in ES:
 - Changes to Demographics Overview screen:
   - Display all address types in top-most pane on screen
   - Add 'Add Address' hyperlink to allow user to add a new address of any type
 - Changes to Add/Edit Address screen:
   - User may select the Address Type from a drop-down list
   - Selecting an Address Type from the drop-down list will automatically 
 load the selected address type on the screen
   - User may Add or Edit a Temporary Mailing address
   - User may Add or Edit a Confidential address (DORMANT)
   - User may Add or Edit a Residential address (DORMANT)
   - User may view current Community Care Determination information on the 
 new Community Care Determination page (DORMANT)
  
 Financial Assessment Means Test
 -Automatically creates a non-disclosure means test for Veterans in pending
 status:  Means Test Required where the eligibility status is verified. The
 Veteran is identified in a text file and a job executes the business rules 
 in the determined eligibility and determined enrolment processes.
