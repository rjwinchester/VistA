Subject: ENROLLMENT SYSTEM (ES) 5.0.1 RELEASE

Category: 
  - Informational

Description:
============

 The purpose of this Informational Patch is to announce the release of the
 Enrollment System (ES) 5.0.1. This release, developed in Java technology,
 contains Enrollment System Modernization (ESM) maintenance defect repairs.
 
 The following functionality is updated in this ES 5.0.1 release:
 Sustainment: 
 1.  Update the Internal Revenue Service (IRS) client interface to repair
     one defect causing 2017 Affordable Care Act (ACA) information returns
     to be rejected by the IRS.

 Rational Team Concert (RTC):
 
 RTC #  Problem/Resolution:
 -----  -------------------
 662918 Problem: All production 2017 ACA information return transmissions to
        the IRS are being rejected

 CoveredIndividualAnnualIndicator from the transmission
