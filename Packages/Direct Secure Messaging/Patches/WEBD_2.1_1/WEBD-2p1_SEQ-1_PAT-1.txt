Subject: Direct Secure Messaging Enhancements Phase 2 (DSMEP2)  Ver. 2.1.1

Category: 
  - Informational

Description:
============

 Direct Secure Messaging (DSM) Release 2.1.1 is a Technical Reference 
 Manual (TRM) and Security Compliance update containing underlying 
 software upgrades for Apache Tomcat, Apache HTTP Web Server, Java Runtime
 Environment, Postfix, NGINX, and Red Hat Enterprise Linux. This build also
 includes an increase to the attachment size limit for Direct messages.
 Several additional minor fixes for legacy defects and 508 compliance
 remediation are also included. These changes are Work Items that were
 recorded and tracked in the Direct Secure Messaging area of Rational Team
 Concert, the VA-prescribed tool used for tracking requirements and
 associated test plans. This release contains the Work Items completed
 during a production deployment. This TRM, security compliance and
 performance enhancements release consists of the following software
 upgrades to the Direct Secure Messaging system:
  
 -Upgrade of Apache Tomcat to v8.5.24
 -Upgrade of Apache HTTP Web Server to v2.4.12
 -Upgrade of Postfix to v3.2.6
 -Replacement of the Java Development Kit (JDK) with 
  Java Runtime Environment (JRE) version 8u161
 -Upgrade of NGINX to v1.14.0  
  
 Supporting technical documentation including the DSM System Design 
 Document, User Guide, Production Operation Manual (POM), and
 Install/Rollback Guide were updated to reflect the integration of the DSS
 provider search capability and the fixes related to the XDM zip file
 issues. 
  
 Test sites: N/A
  
 The upgraded software was deployed to a centralized system at AITC on
 3/21/2019.
