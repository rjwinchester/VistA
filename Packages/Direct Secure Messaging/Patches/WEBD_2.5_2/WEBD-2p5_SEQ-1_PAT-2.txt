Subject: Direct Secure Messaging Phase 3 Version 2.5.0

Category: 
  - Informational

Description:
============

 Direct Secure Messaging (DSM) Release 2.5.0 implements support for the 
 latest clinical care document stylesheet (v2.1), improves application 
 usability via multiple user interface enhancements, upgrades of 
 commercial-off-the-shelf components Java Runtime Environment and Apache 
 Web Server, as well as includes Fortify scan fixes and security 
 remediations. These changes are Work Items that were recorded and tracked 
 in the Direct Secure Messaging area of Rational Team Concert, the 
 VA-prescribed tool used for tracking requirements and associated test 
 plans. This release contains the Work Items completed during a production 
 deployment. 
  
 Supporting technical documentation including the DSM System Design 
 Document, User Guide, Production Operation Manual (POM), and
 Install/Rollback Guide were updated to reflect the above-mentioned 
 application enhancements.  
  
 Test sites: N/A
  
 The upgraded software was deployed to a centralized system at AITC on
 August 27th, 2019.
  
 External Partner Test Sites: 
  
 Partner testing will be performed and verified to be working as expected 
 as part of DSMEP3 Release 2.5.0.
  
 Downstream Dependent System Testing:
  
 DSMEP3 Release 2.5.0 test plan's coverage includes testing efforts with 
 MyHealtheVet, eHealth Exchange, VLER DAS, Master Veteran Index and 
 Veterans Authorizations and Preference.
