Subject: Direct Secure Messaging Enhancements Phase 3 (DSMEP3) Version 2.4.0

Category: 
  - Informational

Description:
============

 Direct Secure Messaging (DSM) Release 2.4.0 includes enhancements to the 
 message composition workflows and graphical user interface to meet the 
 requirements set forth by our business sponsor for the Mission Act and 
 support of the 'opt out' veteran preference model. Also included in this 
 release are several other fixes and application enhancements, including: 
 a technical Reference Manual (TRM) update containing a software upgrades 
 for JQuery, code optimization for the community care document file 
 attachment process, removal of the DSS Provider Directory function from 
 the Compose Message screen, and an update to the data being passed to 
 legacy eHX adapter in preparation for a future HealthShare integration. 
  
 These changes are Work Items that were recorded and tracked in the Direct 
 Secure Messaging area of Rational Team Concert, the VA-prescribed tool 
 used for tracking requirements and associated test plans. This release 
 contains the Work Items completed during a production deployment. 
  
 Supporting technical documentation including the DSM System Design 
 Document, User Guide, Production Operation Manual (POM), and
 Install/Rollback Guide were updated to reflect the above mentioned 
 application enhancements.  
  
 Test sites: N/A
  
 This software was deployed to a centralized system at AITC on June 4, 
 2019

 Downstream Dependent System Testing:
