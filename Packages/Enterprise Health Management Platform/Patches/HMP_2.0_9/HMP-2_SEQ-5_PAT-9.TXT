Subject: EARLY ADOPTER RELEASE OF eHMP USER INTERFACE (UI) V1.2.7

Category: 
  - Informational

Description:
============

 WARNING: This is an Early Adopter version of Enterprise Health Management
 Platform (eHMP) User Interface (UI) version 1.2.7. Only vetted users that
 have completed the eHMP 1.2 training should be given access to eHMP User
 Interface (UI) at this time. 
  
 ***Note*** 
 eHMP technical support staff installed V1.2.7 UI on the eHMP servers at
 the Austin Information Technology Center (AITC), on 12/14/2016.On December
 14, 2016 it was approved for national release to an expanded set of
 approved users across the enterprise.
  
 Only vetted users that have completed the eHMP 1.2 training should be 
 provided access to the user interface (UI). Vetted users must be assigned 
 at least the HMP UI Context Version 0.7-S62 [HMP UI CONTEXT] menu
 option. Users are vetted through the National Data System (NDS), based on
 the list submitted by the sites to VHA eHMP Access Request Team
 (VHAeHMPAccessRequest@domain.ext). This early adopter release of eHMP is not
 yet meant for authoritative clinical use or clinical decision. It is being
 released for comparison and evaluation purposes only. End users are
 encouraged to explore eHMP as opportunities arise throughout their work
 day.
  
 **********************************************************************
 This update provides over 70 bug fixes and performance improvements, most 
 notably a series of improvements to record search. Critical defect fixes 
 include Note text in reports missing from the report in eHMP; Document 
 localTitle field received as numeric field rather than a string field; 
 loading Clinic data once per user session; Disable Operational Data Sync 
 (ODS) check in Resource Development Kit (RDK); Sync Interceptor Clears and
 Re-syncs patients 24 hours after Sync; Fix Order Number Transformation;
 not receiving or seeing Columbia, Missouri reminders; San Antonio, TX
 Production_Problems do not display to eHMP; and eHMP 1.2 Stored Cross-Site
 Scripting (XSS) Vulnerabilities.
