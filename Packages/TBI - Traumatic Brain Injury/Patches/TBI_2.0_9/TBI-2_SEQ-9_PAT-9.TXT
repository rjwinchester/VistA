Description:

 **********************************************************************
 This is to announce that Traumatic Brain Injury Patch TBI*2.0*9 will
 be deployed to the production server for national release on December 21, 
 2015 at 9 p.m. Eastern.
 **********************************************************************
 Note: Technical support staff at the Austin Information Technology Center 
 AITC) will install this patch on the National TBI server located there 
 using a pre-approved Version Description Document(VDD)/Request For Change 
 Order (RFCO) and Implementation Plan (installation/back out 
 instructions).  No effort is required by local sites.  User Acceptance 
 Testing is by direction of the Registry Business Owner 
 and no Memorandum of Understanding (MOU) is required.
 **************************************************************************
 **
 TBI Patch 2.0*9 contains code changes (Enhancements) from New Service 
 Request NSR # 20140109  to the TBI Registry. Changes are Work Items 
 recorded and tracked in the Registries Project Area of RTC. 
 Provide the ability to view summaries of recent patient clinical activity 
 upon confirmation of Patient, Instrument, Note Title and Clinic 
 Appointment in the TBI Instruments Package.
 Provide a consolidated, consistent source of TBI Instrument information 
 that can be viewed for all reporting activities.
 The Reporting Cube functionality will provide the ability to view summary 
 data elements and outcomes, for all standardized forms in the TBI 
 Instruments package.
 TBI Patch 2*9 contains code changes (defect fixes and enhancements) to 
 the TBI Registry.  These changes are Work Items that were recorded and 
 tracked in the Registries Project Area of RTC.
  
 TBI Patch 2*9 includes the TBI Registry Work Items listed below.
  
 158293  Provide the ability to view a Plot Line only if there is a 
 minimum of 2 or more summary scores.
 158286  Provide the ability to view Date, Location, Instrument and 
 Summary Score on each summary line.
 158292  All Plot Lines shall be in Pyramid Analytics.
 158289  Provide the ability to view the full activity (Instrument) 
 associated to a summary line on the complete listing, by selecting that 
 instrument. 
 158300  Allow users to save reports as a template for reuse.
 158304  The report shall be based on the summary score within each report
 158284  Provide the ability to view summaries of recent patient clinical 
 activity upon confirmation of Patient, Instrument, Note Title and Clinic 
 Appointment in the TBI Instruments Package.
 158285  Provide the ability to automatically present a view of summary 
 information from the last three activities.
 158294  The Plot Line reporting shall be based on the summary score 
 within each report.
 158291  Provide the ability to view a Plot Line of the Summary Scores.
 158303  Allow the selection of multiple instruments to report into a 
 single report.
 158302  Provide the ability to view information consolidated from 
 multiple instruments into a single report.
 158298  The Reporting Cube functionality will provide the ability to view 
 summary data elements and outcomes, for all standardized forms in the TBI 
 Instruments package.
 158305  Provide the ability as new instrument packages are developed, to 
 view their information as part of the Reporting Cube functionality.
 158297  Provide the ability as new instrument packages are developed, to 
 view their information as part of the consolidated source of Instrument 
 information.
 158299  Provide the ability to view facility level summary data based on 
 user's preference for keyed variables. (I.E. date, location, gender, age 
 etc.).
 158301  Provide the ability to view each question within each instrument 
 as a value for reporting.
 158290  For all report sections, 'COUNT' refers to a count of unique 
 patients for the given report parameters.
 158288  Provide the ability to view a complete summary list of all prior 
 activity.
 158295  Provide a consolidated, consistent source of TBI Instrument 
 information that can be viewed for all reporting activities
 158296  Provide the ability to view consolidated TBI Instrument 
 information without manually merging tables.
 158287  Provide the ability to view the full activity (Instrument) 
 associated to a summary line by selecting that instrument
