 Patch Components:
 -----------------
 Veteran Authorization and Enhancements (VAP) Release 2.4.0.11 contains 
 code changes (enhancements) to the VAP application.  These changes are 
 Work Items that were recorded and tracked in the Veteran Authorization 
 and Preferences area of Rational Team Concert, the VA-prescribed tool 
 used for tracking requirements and associated test plans.  This release 
 contains the Work Items completed during production deployment.
  
 This enhancement release consists of the following enhancements to the 
 Veteran Authorization and Preferences (VAP) system:
 -       Enhancements to allow users to enter and view comments related to 
 Consent Directives and Individuals
 -       Addition of Veteran  Integrated Service Network (VISN) column to 
 the Consent Directive Detail Report
 -       Enhancements to the Patient Details screen for collapsing and 
 expanding the Status History and Correlations sections
 -       Web Services to allow for the storage and retrieval of documents 
 -       New web services functionality for managing consent directives 
 with trusted VA systems (obtain Veteran's active status, list of 
 partners, list of restrictions)
 -       Updates to Consent Management Application Programming Interface 
 (API) for ensuring consent directive forms can be received and processed 
 by Veteran Authorization and Preferences (VAP) system
 -       Updates to accept interface identification data from external 
 entities and ensure systems leveraging Veteran Authorization and 
 Preferences API are trusted
 -       Enhancements to allow VAP Admins to view a audit report of web 
 service transactions
