Subject: Veteran Authorization and Preferences (VAP) 2.7.0 Release

Category: 
  - Informational

Description:
============

 NOTE: Technical support staff at Austin Information Technology Center 
 (AITC) will install the enhancement release 2.7.0 on the NVP (formerly 
 VLER and formerly NHIN) and NHI servers located there using a 
 pre-approved Request for Change Order and Implementation Plan. This 
 deployment is scheduled to take place on May 16, 2017 from 8:00 PM EST - 
 11:00 PM EST. No effort is required by local sites.  
  
 Veteran Authorization and Enhancements (VAP) Release 2.7.0 contains 
 enhancements to the VAP application and eHealth Exchange (eHX). These 
 changes are Work Items that were recorded and tracked in the Veteran 
 Authorization and Preferences area of Rational Team Concert, the 
 VA-prescribed tool used for tracking requirements and associated test 
 plans. This release contains the Work Items completed during production 
 deployment.
  
 This release consists of the following enhancements:
 -       Improved accessibility and usability through interface usability 
 improvements for Graphical User Interfaces (GUIs) and modernization of 
 the page layout
 -       Provides users the ability to view raw XML files associated with 
 C32 and C62 disclosures from the Disclosure Report
 -       A check is added for VAP to use the latest CCDA stylesheets and 
 for C62s to be generated in PDF format for users to access
 -       Modification to display the restricted organizations on the 
 Consent Directive report 
 -       Updated VAP facility tracking to provide support for integrated 
 VISNs, creating an interface where the user can modify facilities and 
 facility information 
 -       Partner description field added to the Partner Management page
  
 User Acceptance Test (UAT) is by the direction of the VAP Business Owner 
 and no Memorandum of Understandings (MOUs) are required. 
  
 Testing Process: Release 2.7.0 was fully tested though the Office of 
 Information and Technology (OIT) Software Quality Assurance. The VAP 
 Business Owner and Stakeholders completed UAT of enhancements. 
  
 Downstream Dependently System Testing: Testing was done with eHX, 
 interfacing system.
