Subject: VLER VAP Enhancements Phase 2 Build 2 (v.3.1.0.7)

Category: 
  - Informational

Description:
============

 Note: Technical support staff at Austin Information Technology Center 
 (AITC) will install this enhancement release on the VAP servers located 
 there using a pre-approved Request for Change Order and Implementation 
 Plan.  This Deployment is scheduled to take place on June 19, 2018 
 from 8:00 PM EST- 11:00 PM EST.
 No effort is required by local sites.  User Acceptance Test is by the 
 direction of the VAP Business Owner and no MOUs are required.
 **************************************************************************
 VAP Release 3.1.0.7 contains code changes (enhancements) to the VAP 
 application and service.  These changes are Work Items that were recorded 
 and tracked in the VAP area of Rational Team Concert, the VA-prescribed 
 tool used for tracking requirements and associated test plans.  This 
 release contains the Work Items completed during production deployment.
 This enhancement release consists of the following enhancements to the 
 VAP system:
 -       Ability to schedule exported reports in advance
 -       Display of Consolidated Clinical Document Architecture (C-CDA) 
 Continuity of Care Document (CCD) Release 2.1 in VAP
 -       Ability to delete pending scheduled reports
 -       Information displayed within the dashboards under the Summary 
 navigation tab of the VAP application is now accessible and the content 
 of the information is now conveyed using a screen reader and keyboard 
 -       Users requiring the use of accessibility tools can access Print 
 and Mail Icons on the Status History table
 -       The tooltips used to provide readers with additional content on 
 form fields and columns are now accessible using a screen reader and 
 keyboard 
 -       The messages that appear on the user interface to the users when 
 a report is still loading are available to a non-sighted user using a 
 screen reader.  In addition, when the report has finished loading, the 
 screen reader alerts non-sighted users that the report has completed 
 processing. 
 -       The expand/collapse buttons on all the VAP application pages are 
 now operable using a keyboard and the state of the button is now read 
 when using accessibility tools
 -       508 Compliance: Expand & Collapse Buttons
  
 Test sites: N/A
 The upgraded software will be deployed to a centralized system at AITC.
  
 Downstream Dependent System Testing: Testing efforts with VAP downstream 
 systems were successful.
