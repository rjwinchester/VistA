Description:
============

  
 This is the informational patch for the release of version 5.3 of the 
 Voluntary Service System (VSS) Web application.
  
 Release Information:
 ======================
 VSS is an enterprise web application that allows VA Voluntary Service
 (VAVS) staff to track Volunteers, Donors and Donations to VA 
 organizations.  VSS*5*3 implements additional enhancements to the system 
 to better track and report volunteer assignments and how donated 
 dollars are allocated.
  
 Software Changes:
 =====================
 VSS changes in 5.3 will be deployed to the production servers and will be 
 immediately available to users after deployment.  No changes are required 
 at the local sites.
  
 Enhancements and New Features
 =============================
 1. Merge Branches and Organizations: VSS users can merge organizations 
 and associated branches in order for all relevant information to be 
 under one entity.
  
 2. Meal ticket Print: When clicking on the Print Selected button on the 
 Meal Ticket Screen, the new Format Select box gives the user the 
 option to print from a normal or receipt (kiosk) printer.
  
 3. The Manage Letter Template Type drop-down is revised for improved 
 readability. The following options now available are:
         Monetary - Not in Memory Of
         Monetary - In Memory Of
         Family Contact (To Donor)
         Family Contact (To Family)
         Activity Donation
         Item Donation
  
 Reports
 -----------------------
 1. AJR Report: Users have the ability to create a customized Annual
 Joint Review(AJR) Report and capture the AJR Response(AJR 10-1240a).
  
 2. Daily Volunteer List: allows user to view a detailed history of all 
 volunteers' hours within a date range and the date and time of the 
 entry, as well as the user that made the last change.
  
 3. NAC Organization Yearly Comparison: The NAC Yearly Comparison Report 
 Compares the total number of volunteers from each NAC organization by
 Facility five years back from the defined date. 
  
 4. National Organization Affiliated Volunteers Report: Compares the total 
 number of regular and occasional volunteers from each NAC     
 organization by Facility, State, or VISN from five years back from the 
 defined date.  
  
 5. Volunteer Daily Review: Enables users to run a report on a single 
 volunteer and view a detailed history of the volunteer's hours.
  
 6. Volunteers by Service and Facility: Displays the number of volunteers 
 for the selected facility(s) by service within number of months 
 specified.   
  
 7. Volunteers By Service:  Displays detail and contact information 
 related to the Voluntary Service office at each facility.
  
 8. The Committee Listing Report is updated to include the committee 
 attendance members with assignments, regardless if they have no hours 
 logged or visits for the fiscal year selected. If none are recorded,
 the volunteer will be listed on the report with the hours of 0.00 in 
 Quarter Hours.
