Subject: BMS Version 2.0.2

Category: 
  - Informational

Description:
============

 The Bed Management Solution (BMS) v2 software enhancement
 project is to improve usability of the BMS v2 system, which improves
 the data accessibility and usability for BMS v2 users, and
 increase the integration of BMS v2 and VA systems.
  
 The following enhancements are included in this release:
  
 1. BMS PIV/Single Sign On (SSO) Capability:
         The Bed Management System (BMS) application now has the ability
         to allow PIV Card authentication.  When accessing BMS via the
         website, the user will be redirected to the VA Single Sign-On
         portal.  Users are presented the option of Signing In with their
         VA PIV Card or other Sign-In Options.
 2. Fortify Security vulnerability remediation:
         The Fortify issues as identified by Rule Pack 3 of the Fortify
         Tool have been resolved with this release.
 3. 508 Compliancy:
         All issues as previously identified by the 508 Team have been
         resolved with this release.

 Installation Instructions:
 --------------------------
 The BMS application is a centrally managed web-based application
 hosted at AITC.
 This patch will be loaded into BMS Production account
 by AITC staff.
  
 No software will be installed at the facilities.
  
 Reporting Problems:
 -------------------
 If there are problems associated with the BMS application they should be
 reported via CA Service Desk by contacting the National Service Desk (NSD)
 at 1-855-673-4357 for assistance.
