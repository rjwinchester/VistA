Subject: BMS v2.3 VIA Migration Release

Category: 
  - Informational

Description:
============

 The Bed Management Solution (BMS) v2 software enhancement
 project is to improve usability of the BMS v2 system, which improves
 the data accessibility and usability for BMS v2 users, and
 increase the integration of BMS v2 and VA systems.
  
 The following enhancements are included in this release:
  
 BMS VIA Migration capability:
 As part of a congressional mandate,
 the BMS Team is issuing a Patch on 5/24/2018 to modify the
 interface connection to VistA. This release will remove the
 MDWS connection and replace it with the VistA Integration 
 Adapter (VIA) connection.
