Subject: VBECS 2.3.1.1 Rev B Informational Patch

Category: 
  - Informational

Description:
============

 This information patch (VBEC*2*4) serves to alert VA facilities using the 
 Vista Blood Establishment Computer Software (VBECS) that VBECS 2.3.1.1 
 Rev B server patch has been released for installation.
  
 Sites who have not installed VBECS have no action to take with this 
 informational patch.
  
 A full description of patch changes and implementation process are 
 included in the VBECS 2.3.1.1 Rev B Release Notes.
  
 The Test and Production accounts will be installed separately. There is 
 no downtime associated with this patch installation. Refer to the VBECS 
 2.3.1.1 Rev B Release Notes for further installation details.
  
 Items addressed in VBECS 2.3.1.1 Rev B
 1.      Regulatory Compliance Updates
   1.1.     Blood Product updates released by ICCBBA November 2018
            through February 2019
   1.2.     Blood product expiration time updates.
   1.3.     Blood product short name updates.
 2.      Customer Reported Issues and Enhancements
   2.1.     Fix known defects and requests related to the Division
            Workload Report.
   2.2.     Fix known defects and requests related to Modify Units.
   2.3.     Fix known defects and requests related to the Inappropriate
            Transfusion Report.
   2.4.     Fix known defects and requests related to the Patient
            Search Utility.
  
 Refer to the VBECS 2.3.1.1 Rev B Release Notes for more details on the 
 changes included in this revision letter patch.
  
 BLOOD BANK CLEARANCE
 ====================
 EFFECT ON BLOOD BANK FUNCTIONAL REQUIREMENTS: Patch VBEC*2*4 contains 
 changes to a package referenced in VHA OI SEPG SOP 192-023 Review of 
 VISTA Patches for Effects on VISTA Blood Bank Software. This patch does 
 not alter or modify any VistA Blood Bank software design safeguards or 
 safety critical elements functions.
  
 RISK ANALYSIS: Changes made by patch VBEC*2*4 have no effect on VistA 
 Blood Bank software functionality, therefore RISK is none.
  
 VALIDATION REQUIREMENTS:
 ========================
 The VBECS 2.3.1.1 Rev B server patch requires customer validation in the 
 local Test account prior to installation in Production to meet Food & 
 Drug Administration (FDA) requirements. The user validates the Blood Bank 
 software for its intended use prior to its acceptance and use for patient 
 care. The VBECS 2.3.1.1 Rev B Release Notes contain descriptions of the 
 changes made in this patch and includes validation scenarios.
  
 More specific details on the installation schedule will be included in a 
 separate VBECS ListServ message. Please review that schedule to assess 
 your impact analysis and testing of the changes prior to installation in 
 your Production VBECS. To sign up for VBECS ListServ messages, refer to 
 the VBECS SharePoint site at:
 https://vaww.oed.portal.domain.ext/projects/vbecs/default.aspx
