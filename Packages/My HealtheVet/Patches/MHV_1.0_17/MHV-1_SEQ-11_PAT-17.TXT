 Infrastructure & Interface (I&I) Team Enhancements
 ==================================================
  
 > MHV Admin Portal Enhancements
  
 ----- Help Desk Administrator can resolve multiple Correlations in MVI by 
 using the new "Remove Correlation in MVI" action button.
 ----- The Correlate Matched account(s) Background Job is up and running.
 ----- Display Match button for stuck accounts
  
 -> MHV Automated Deactivation for A43 Patient Move Message:
  
 ----- The MHV user account that was successfully deactivated has the word 
 "DELETED" appended to the last name if an A43 Patient Move              
 message was received. 
 ----- The MHV System displays an error message when the deactivated user 
 attempts to log into the MHV System.
 ----- The Help Desk Administrator can view the reason, description and 
 date of the deactivation in the user details page under the             
 Help Desk page.
  
 -> MHV Treatment Facility Notification:
  
 ----- Treatment Facility list for users with Advanced and Premium 
 accounts will be updated at the time of login and through the           
 monthly batch update process.
  
 -> MHV Account Upgrade using eRAR Form:
  
 ----- User should be able to upgrade Account online using the new flow as 
 depicted in US01 and should see the updated page content.
  
 -> MHV Blue Button (BB) Continuity of Care Document (CCD) via Direct:
  
 ----- User should be able to see Account Activity Log (AAL) updated to 
 include receipt of successful or unsuccessful delivery of the           
 Health Summary (VA CCD) 
 ----- User should be able to see the summary table for Send My Health 
 Summary (VA CCD) information in user interface.
 ----- The Health Summary (VA CCD) file is sent to one DIRECT email 
 address.
  
 -> MHV IRIS (Inquiry Routing and Information System) Integration:
  
 ----- The User has successfully submitted a Help Desk inquiry:
         ----- Selecting the Contact Help Desk Image/Link or "This Form" 
 Text Link - Logged On
         ----- Selecting the Contact Help Desk Image/Link or "This Form" 
 Text Link - Not Logged On
 ----- Contact Help Desk by using the Technical Support link from within 
 Secure Messaging.
 ----- Selecting the Contact Us Link from MHV Admin Portal Home Page and 
 from within Secure messaging page.
 ----- The user can see an error message when IRIS is not accessible.
